# ✈️ AI Airline Sentiment Analyzer

A Streamlit application that predicts whether an airline-related tweet is **Positive**, **Neutral**, or **Negative** using a Deep Learning model and word embeddings.

## Features

- ✈️ Professional airline-themed UI
- 🤖 TensorFlow Deep Learning model
- 📝 Tweet sentiment prediction
- 📊 Confidence score
- 📈 Prediction probabilities

## Project Structure

```
app.py
sentiment_model.keras
embedding_lookup.pkl
scaler.pkl
label_encoder.pkl
requirements.txt
README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
